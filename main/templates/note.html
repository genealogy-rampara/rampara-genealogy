જ્યારે ડેટા CSV મા ઉમેરવો હોય ત્યારે ધ્યાન મા રાખવી આ વિગતો

ડેટા કાઢવો નહીં !

Follow instructions for correct data format (સાચા ડેટા ફોર્મેટ માટે સૂચનાઓને અનુસરો) :
૧. ID, child_id, નામ, જાતિ, બાપુ સાહેબ, બા સાહેબ, કુંવરસાહેબ / કુંવરીબા, રાણી સાહેબ / જમાઈ સાહેબ, રાણી સાહેબ ના બાપુ સાહેબ નું નામ / જમાઈ સાહેબ ના બાપુ સાહેબ નું નામ, રાણી સાહેબ નું વતન / જમાઈ સાહેબ નું વતન.
૨. ID અને child_id ને ઈંગ્લિશ મા જ લખવી. (ID અને child_id અલગ - અલગ હોવી જોઈએ).
૩. ID અને child_id પેહલા થી CSV માં હોવી જોઈએ નહીં.
૪. Name પેહલા થી CSV માં હોવી જોઈએ નહીં અને જો હોય તોહ Name ના પાછળ સ્પેસ લગાવો.
૫. child_id મા વ્યક્તિ જે ID મા હોય તે લખવી (નીચે દર્શાવેલ છે).
૬. જન્મ તારીખ ના ખબર હોય તો ખાલી છોડી દેવુ.
૭. જાતિ મા પુરુષ માટે M લખવું અને સ્ત્રી માટે F લખવું.
૮. ડેટા ઉમેરિયા બાદ થોડી રાહ જોવી.
૯. જો ડેટા અપડેટ ના થયો હોય તોહ રિફ્રેશ કરવું.
